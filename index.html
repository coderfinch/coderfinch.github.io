FinchEdit

Experience the innovative code editor FinchEdit. From software flow analysis to intuitive user interface, FinchEdit provides a comprehensive development environment. The editor provides syntax highlighting, code assist and deeper insights into data flow. Integrated with DefChecker, a static source code analyzer tool to enhance software quality.

UI Views

Text View Tab

User can open a .c file from local file system for editing, fix errors and save changes back to filesystem. Syntax highlighting and view parameter info, member lists for function/variable declaration.

Flow Graph View Tab

The Flow Graph View shows the control flow of a selected function as basic blocks and edges. By default, the last function in the .c file currently opened in Text View tab is selected.

Output View Tab

The Output View shows the list of undefined variables observed in .c file currently opened in Text View.

Analysis

Data Flow analysis

Flow sensitive analysis is performed on source code of .c file and included headers. The analysis runs on each function in a translation unit and outputs uninitialized variables.

Pointer-Alias analysis

The points-to analysis is performed to find uninitialized/undefined pointers. Interprocedural analysis is performed for call sites that resolve to functions in the same translation unit.

How it works?

1. Open the folder that has .c/.h files in File Explorer tab.
2. Open the source file to be analyzed in Text View tab.
3. Click Run from menu
4. View the results in Output View tab.
5. Switch to Flow Graph View tab to view the flow graph.
1. Click menuitem File->Open Sample->Open Sample 1
2. Text View tab shows contents of selected sample
3. Switch to Flow Graph View tab to view Control Flow Graph
4. Click menuitem File->Run
5. Output View shows the list of undefined variables.

Prerequisites

1. Install Microsoft Visual C++ 2015-2022 Redistributable (x64).
2. Qt 6.8.3 (using LGPL license).
Source code is available at https://www.qt.io/development/download-open-source The libraries are packaged along with the application. Built using Visual Studio 2022 (64-bit) version 17.14.15 and Qt 6.8.3.
